# Selamat Ulang Tahun, Justin — Website Hadiah

Website ini adalah buku digital berisi perjalanan cerita, dengan tema
"Batman Inspired Luxury" (nuansa Gotham malam hari, tanpa aset berhak cipta Batman).

## Cara Pakai (Sebelum Upload)

1. **Ganti foto**
   Buka folder `images/`, baca `README.txt` di dalamnya, lalu masukkan foto kamu
   (nama file bebas, tulis daftarnya di `photos.txt`).

2. **Ganti video**
   Buka folder `videos/`, baca `README.txt` di dalamnya, lalu masukkan video kamu
   (nama file bebas, tulis daftarnya di `videos.txt`). Opsional, boleh dikosongkan.

3. **Ganti musik**
   Buka folder `music/`, masukkan file lagu dengan nama `birthday.mp3`.

4. **Cek langsung di laptop**
   Buka file `index.html` dua kali klik, atau drag ke browser (Chrome/Edge/Firefox).
   Semua fitur sudah bisa dicoba tanpa install apa pun.

## Cara Upload ke GitHub Pages

1. Buat repository baru di GitHub, misalnya beri nama `birthday-justin`.
2. Upload SEMUA isi folder ini (index.html, style.css, script.js, folder images, folder music)
   ke repository tersebut lewat tombol "Add file" → "Upload files" di GitHub,
   atau lewat GitHub Desktop / git command line.
3. Masuk ke menu **Settings** → **Pages** di repository kamu.
4. Di bagian **Branch**, pilih `main` (atau `master`) dan folder `/root`, lalu klik **Save**.
5. Tunggu sekitar 1-2 menit, GitHub akan memberi link seperti:
   `https://namakamu.github.io/birthday-justin/`
6. Link itu yang kamu kirim ke Justin. Selesai.

## Catatan Teknis

- Website ini murni HTML, CSS, dan JavaScript. Tidak butuh instalasi apa pun.
- Kalau ada foto yang belum diisi, website tetap berjalan normal (otomatis dilewati).
- Kalau musik belum diisi, website tetap berjalan normal tanpa suara.
- Semua sudah responsif untuk HP, tablet, dan laptop.
